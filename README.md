# StudyLink 📚

A social study networking app that connects students to learn, collaborate, and grow together.

## 🌐 Live App
[studylink-app.netlify.app](https://studylink-app.netlify.app)

## ✨ Features
- 💬 Real-time group chat and messaging
- 📝 Community feed for study posts
- 🔍 Find study partners by subject
- 📁 File and media sharing
- 🎙️ Voice messaging
- 🌙 Dark/Light mode
- 🔔 Real-time notifications

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Firebase (Authentication + Firestore)
- Cloudinary (Media uploads)
- Netlify (Hosting)

## 📁 Project Structure
```
studylink/

├── css/
```
